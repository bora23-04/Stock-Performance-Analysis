# Stock Performance Analysis Dashboard

## Project Summary
This project focuses on analyzing the stock performance of major IT companies — TCS, Infosys, and HCL using historical market data. The objective was to provide clear insights into returns, risk, and overall market behavior through an interactive dashboard.

## Objectives
- Analyze stock price trends over the last 2-3 years  
- Compare performance across companies  
- Evaluate risk vs return relationship  
- Identify short-term and long-term trends  

## Tools & Technologies
- Python (Pandas, yfinance)  
- Power BI  
- Data Cleaning & Feature Engineering  

## Key Analysis Performed
- Daily Return Calculation  
- Moving Averages (7-day trend)  
- Volatility Analysis (High-Low %)  
- Cumulative Return (Growth of Investment)  
- Monthly Return Patterns  
- Risk vs Return Comparison  

## Dashboard Features
- Interactive stock comparison  
- Price vs Moving Average visualization  
- Risk vs Return scatter analysis  
- Monthly performance trends  
- Key KPIs (Return, Volatility, Volume)  

## Key Insights
- HCL delivers higher returns but with increased volatility  
- TCS shows more stability with lower overall growth  
- No clear relationship between risk and return is observed  
- Market behavior remains unpredictable with occasional spikes  

##  Data Source
Data collected from Yahoo Finance (last 3 years)
